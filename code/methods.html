Proposed Methods

Design: Quasi-experimental, between-group comparison (existing TIC-trained vs. not trained).

Population: Juvenile justice professionals (e.g., probation, case managers, residential staff).

Setting: Youth detention/residential centers (capstone context originally near Gillette, Wyoming).

Proposed implementation flow (quasi-experimental).

Variables & Measurement

  • Independent variable: TIC training completion (yes/no; training dose can be added later).
  • Dependent variable: Burnout (MBI subscales: emotional exhaustion, depersonalization, personal accomplishment).
  • Covariates (recommended upgrades): tenure, role, shift, caseload intensity, critical incident exposure, supervision quality.

Analysis plan

  • Two-tailed independent samples t-tests for between-group differences.
  • Assumption checks: normality and homogeneity of variance.
  • Recommended Ivy upgrade: report effect sizes and confidence intervals; consider regression adjustment with covariates.
