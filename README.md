# TIC × Burnout Research Microsite

This is a lightweight Ivy-style research microsite (single-page app) for a capstone proposal on Trauma-Informed Care (TIC) and burnout among juvenile justice professionals.

## Publish with GitHub Pages
1. Create a new GitHub repository (e.g., `tic-burnout-research`).
2. Upload all files from this folder to the repo root.
3. In GitHub: Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: `main` / root
4. Wait for the Pages URL, then open it.

## Customize
- Edit `sections/*.html` to update content.
- Replace diagrams in `assets/diagrams/`.
- Add a PDF to `assets/` and link it from `sections/download.html`.

## Integrity note
Figures labeled “illustrative” are conceptual only (not empirical results).
